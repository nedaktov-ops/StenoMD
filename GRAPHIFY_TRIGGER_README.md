# GraphifyStenoMD Trigger

## Usage

When working on your StenoMD project in OpenCode, type:

```
use Graphify [CONTEXT]
```

OpenCode will automatically determine the best action based on your context.

## Quick Examples

```
use Graphify analyze gaps        # Find missing data
use Graphify query Vasile       # Find politician
use Graphify check health      # Validate data
use Graphify report activity  # Generate report
use Graphify validate           # Full health check
use Graphify daily             # Daily workflow
```

## Full Trigger Phrase

The magic phrase is **`use Graphify`** - OpenCode recognizes it and executes the appropriate GraphifyStenoMD workflow.
