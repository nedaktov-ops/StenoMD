# Parliamentary Reference Index

Reference materials for understanding Romanian Parliament.

## Directory Structure

```
_parliament/
├── constitutional/     - Constitutional articles (58-79, 113-114)
├── definitions/      - Key parliamentary terms
├── committees/       - 44 committee structures
├── process/          - Legislative process flow
└── legislatures/     - All legislatures since 1990
```

## Quick Reference

### Law Types
- Constitutional: 2/3 majority
- Organic: majority of all members
- Ordinary: majority of present

### Initiative Types
- Government: Proiect de lege
- Parliamentary: Propunere legislativă
- Citizens': Inițiativă cetățenească (100K+ signatures)

### Key Articles
- Art. 58-71: Parliament organization
- Art. 72-78: Legislative process
- Art. 113-114: Confidence/Censorship

### Committees
- Senate: 15 standing
- Chamber: 17 standing
- Joint: ~12

### Process Flow
Initiation → Committee → Plenary → Second Chamber → Promulgation → Monitorul Oficial → Effect
