## Brain Model

The vault is organized to mirror brain functions:

### Sensory Input
- Source URLs tracked
- Data sources listed
- Last sync timestamps

### Processing  
- Activity scores calculated
- Collaboration analyzed
- Party alignment mapped

### Memory
- Proposals linked
- Speeches indexed
- Sessions stored

### Action/Output
- Dataview queries ready
- Alerts generated
- Summaries auto-created
