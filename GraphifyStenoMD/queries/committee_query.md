# Committee Query

**Trigger:** `/stenomd xquery committee`
**Purpose:** Find committee members and activity

---

## Usage

```
/stenomd xquery committee buget
/stenomd xquery committee justitie
/stenomd xquery committee all
```

---

## Example

```
/stenomd xquery committee buget

=== Committee: Buget, Finanţe ===

Members: 23
Chair: (from graph)
Vice-Chairs: 2

Party Distribution:
- PSD: 12
- PNL: 8
- AUR: 3
```
