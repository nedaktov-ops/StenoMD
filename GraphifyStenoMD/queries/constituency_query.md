# Constituency Query

**Trigger:** `/stenomd xquery constituency`
**Purpose:** Find politicians by constituency

---

## Usage

```
/stenomd xquery constituency Bucharest
/stenomd xquery constituency Dolj
/stenomd xquery constituency all
```

---

## Example

```
/stenomd xquery constituency Dolj

=== Constituency: Dolj ===

Deputies: 5
Senators: 3

Members:
- TĂNĂSESCU Alina-Elena (senator, PSD)
- (4 deputies)

Activity:
- Total speeches: 234
- Sessions: 45
- Laws proposed: 12
```
