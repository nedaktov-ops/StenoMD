# Party Analysis Query

**Trigger:** `/stenomd xquery party`
**Purpose:** Analyze party activity and membership

---

## Usage

```
/stenomd xquery party PSD
/stenomd xquery party PNL
/stenomd xquery party activity
```

---

## Example Output

```
=== Party: PSD ===

Members: 142 (89 deputies, 53 senators)

Activity:
- Total speeches: 4,523
- Laws proposed: 234
- Sessions: 892

Top 5 Speakers:
1. ANDREI DANIEL GHEORGHE (87)
2. ANAMARIA GAVRILĂ (72)
...

Cross-Party Collaborations:

```
