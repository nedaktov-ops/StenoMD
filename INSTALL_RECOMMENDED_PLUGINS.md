# Recommended Obsidian Plugins for StenoMD

These community plugins enhance the StenoMD vault experience. Install them via Obsidian's Community Plugins browser.

## Essential (Highly Recommended)

### 1. Dataview ⭐⭐⭐⭐⭐
**Why:** Powers all dynamic queries in the vault (MP rankings, session lists, law tracking).

**Install:** Settings → Community plugins → Browse → Search "Dataview" → Install

**Usage:** 
- Queries in `vault/_brain/queries/` rely on Dataview
- Example: `deputy-recall.md`, `law-recall.md`

**Note:** Already required - please install to use query-brain.py and dynamic views.

---

### 2. Metadata Menu ⭐⭐⭐⭐⭐
**Why:** Edit frontmatter fields visually without YAML editing.

**Install:** Search "Metadata Menu" in Community plugins

**Usage:**
- Open any deputy/session/law note
- Click the metadata icon in the ribbon
- Edit fields in table form

---

### 3. Advanced Tables ⭐⭐⭐⭐
**Why:** Better table editing, sorting, alignment.

**Install:** Search "Advanced Tables"

**Usage:**
- Improves editing of any markdown tables
- Useful for committee assignments tables

---

## Very Useful

### 4. Calendar
**Why:** Navigate sessions by date, see weekly/monthly views.

**Install:** Search "Calendar"

**Configuration:**
- Set week start to Monday
- Link calendar to `vault/sessions/` folder

---

### 5. Tasks
**Why:** Integrates with the task tracking system.

**Install:** Search "Tasks"

**Configuration:**
- Set default folder to `vault/tasks/` (if used)

---

### 6. QuickAdd
**Why:** Rapidly create new deputy/session/law notes with templates.

**Install:** Search "QuickAdd"

**Setup:**
- Create scripts for: new deputy, new session, new law
- Use templates from `vault/_templates/`

---

### 7. Note Refactor
**Why:** Bulk rename, move, merge notes (useful after data updates).

**Install:** Search "Note Refactor"

---

### 8. Copy as Link
**Why:** Enhanced wikilink copying (relative paths, aliases).

**Install:** Search "Copy as Link"

---

## Optional

### 9. Outliner
**Why:** Better heading navigation and restructuring.

### 10. Tag Focus
**Why:** Filter vault by tags (usable for filtering by party, committee, etc.)

---

## Installation Steps

1. Open Obsidian with StenoMD vault
2. Go to Settings → Community plugins
3. Turn on "Community plugins" if prompted
4. Click "Browse" and search for each plugin
5. Click "Install" then "Enable"
6. Adjust settings per recommendations above

---

## Plugins Already Configured (Core)

The following Obsidian core plugins are already enabled in `.obsidian/core-plugins.json`:

- File explorer, Global search, Switcher, Graph, Backlinks, Canvas, Outgoing link, Tag pane, Properties, Page preview, Daily notes, Templates, Word count, Outline, etc.
